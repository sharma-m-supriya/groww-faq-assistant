 📈 Groww FAQ Assistant

A smart FAQ chatbot built using Streamlit that answers factual mutual fund queries with verified sources.  
Inspired by the clean UI and experience of Groww.

---

Features

- 🤖 **Smart Chatbot Search**  
  Uses semantic search (Sentence Transformers) to understand user queries.

- 📚 **Factual Answers Only**  
  Provides verified information about mutual funds (no advice).

- 🔗 **Clickable Source Links**  
  Every answer includes a trusted source.

- 💡 **Suggested Questions**  
  Helps users understand what they can ask.

- 🎯 **Clean UI**  
  Minimal, modern interface inspired by Groww.

- ⚠️ **Advice Rejection**  
  Blocks investment advice queries like "what should I invest in?"

---

Tech Stack

- Python
- Streamlit
- Sentence Transformers
- Scikit-learn
- Pandas
